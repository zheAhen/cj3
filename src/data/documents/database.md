# 数据库系统概论

## 课程概述
本课程是数据库领域的入门课程，适合计算机及相关专业学生。

## 主要内容
- [[关系数据库]]：关系模型、SQL语言
- [[数据库设计]]：ER模型、范式理论
- [[事务处理]]：ACID特性、并发控制
- [[数据库新技术]]：NoSQL、大数据

## 评价
理论与实践结合紧密，通过实际项目加深对数据库的理解。
