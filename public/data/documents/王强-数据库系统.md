# 王强-数据库系统

## 课程概述
本课程是计算机科学专业的重要课程，全面讲解数据库系统原理、设计方法和实现技术，包括关系模型、SQL语言、数据库设计等内容。

## 课程特色
1. **理论与实践结合**：配套数据库实验课程
2. **项目驱动**：完成完整的数据库应用系统开发
3. **前沿技术**：介绍NoSQL、大数据等新技术
4. **考核方式**：实验作业(40%)+期中考试(20%)+期末项目(40%)
