# 林敏-环境经济学

## 课程概述
本课程是环境科学专业的交叉课程，研究环境问题与经济系统的关系，包括环境政策、资源定价、可持续发展等内容。

## 课程特色
1. **政策分析**：评估环境政策效果
2. **案例研究**：分析环境经济案例
3. **模型构建**：建立环境经济模型
4. **考核方式**：政策分析(50%)+期中考试(30%)+课堂讨论(20%)
