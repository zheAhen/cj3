# 李华-微观经济学

## 课程概述
本课程是经济学专业的基础课程，系统讲解微观经济学理论框架，包括供需理论、消费者行为、生产者行为、市场结构等内容。

## 课程特色
1. **理论联系实际**：结合当前经济现象分析
2. **数学工具应用**：适度运用数学推导
3. **课堂讨论**：鼓励学生参与经济问题讨论
4. **考核方式**：平时作业(30%)+期中考试(30%)+期末考试(40%)
