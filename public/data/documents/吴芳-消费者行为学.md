# 吴芳-消费者行为学

## 课程概述
本课程是市场营销专业的重要课程，研究消费者决策过程和行为模式，包括感知、学习、态度形成、购买决策等内容。

## 课程特色
1. **实验研究**：设计并实施消费者行为实验
2. **市场调研**：开展实际消费者调查
3. **行为观察**：记录分析消费者购物行为
4. **考核方式**：调研报告(50%)+期中考试(20%)+期末论文(30%)
