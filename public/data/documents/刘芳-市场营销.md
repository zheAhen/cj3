# 刘芳-市场营销

## 课程概述
本课程是工商管理专业的核心课程，系统讲解市场营销的基本理论、策略和实践技巧，包括市场细分、目标市场选择、定位策略等内容。

## 课程特色
1. **案例教学**：分析国内外知名企业营销案例
2. **实战项目**：分组完成真实企业营销策划
3. **嘉宾讲座**：邀请企业营销高管分享经验
4. **考核方式**：小组项目(50%)+个人报告(30%)+课堂参与(20%)
