# 赵敏-金融学

## 课程概述
本课程是金融学专业的核心课程，系统讲解金融市场、金融机构和金融工具的基本原理，包括货币时间价值、风险与收益、资产定价等内容。

## 课程特色
1. **数据分析**：使用真实金融数据进行分析
2. **模拟交易**：组织股票模拟交易比赛
3. **行业洞察**：邀请金融从业者分享经验
4. **考核方式**：案例分析(40%)+期中考试(30%)+期末论文(30%)
