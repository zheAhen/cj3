# 周伟-商业分析

## 课程概述
本课程是商业分析专业的核心课程，讲解如何利用数据分析支持商业决策，包括数据收集、分析方法、可视化呈现等内容。

## 课程特色
1. **案例实战**：分析真实商业案例
2. **工具应用**：掌握Power BI等分析工具
3. **决策模拟**：基于数据做出商业决策
4. **考核方式**：分析报告(60%)+期中考试(20%)+课堂展示(20%)
