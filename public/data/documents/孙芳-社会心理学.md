# 孙芳-社会心理学

## 课程概述
本课程是心理学专业的重要课程，研究个体在社会环境中的心理和行为，包括社会认知、态度、群体行为、人际关系等内容。

## 课程特色
1. **实验研究**：设计社会心理学实验
2. **社会观察**：记录分析社会现象
3. **角色扮演**：模拟社会情境
4. **考核方式**：研究报告(50%)+期中考试(30%)+课堂参与(20%)
