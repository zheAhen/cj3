# 郑强-计算机网络

## 课程概述
本课程是计算机科学专业的核心课程，系统讲解计算机网络的基本原理和协议，包括网络体系结构、TCP/IP协议、网络安全等内容。

## 课程特色
1. **实验操作**：配置路由器和交换机
2. **协议分析**：使用Wireshark分析网络流量
3. **项目实践**：设计小型网络系统
4. **考核方式**：实验报告(40%)+期中考试(30%)+期末项目(30%)
