# 赵刚-操作系统

## 课程概述
本课程是计算机科学专业的核心课程，深入讲解操作系统的基本原理和实现技术，包括进程管理、内存管理、文件系统、设备驱动等内容。

## 课程特色
1. **实验操作**：实现小型操作系统模块
2. **源码分析**：研究Linux内核源码
3. **性能优化**：优化系统性能实验
4. **考核方式**：实验报告(50%)+期中考试(30%)+期末项目(20%)
