# 李敏-数据可视化

## 课程概述
本课程是数据科学专业的重要课程，讲解数据可视化的原理和方法，包括图表设计、交互可视化、可视化工具应用等内容。

## 课程特色
1. **项目实践**：完成多个可视化项目
2. **工具应用**：掌握Tableau、D3.js等工具
3. **设计思维**：培养可视化设计能力
4. **考核方式**：项目作品(60%)+期中考试(20%)+课堂展示(20%)
