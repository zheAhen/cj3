# 周敏-财务管理

## 课程概述
本课程是会计学专业的核心课程，讲解企业财务管理的基本理论和方法，包括财务分析、投资决策、融资决策等内容。

## 课程特色
1. **案例分析**：分析上市公司财务报表
2. **估值实践**：对企业价值进行评估
3. **Excel应用**：使用Excel进行财务建模
4. **考核方式**：案例分析(50%)+期中考试(20%)+期末考试(30%)
