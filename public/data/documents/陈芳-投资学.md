# 陈芳-投资学

## 课程概述
本课程是金融学专业的核心课程，系统讲解投资理论和实务，包括资产定价、投资组合、风险管理、行为金融等内容。

## 课程特色
1. **模拟投资**：组织股票模拟交易比赛
2. **数据分析**：使用真实金融市场数据
3. **案例研究**：分析经典投资案例
4. **考核方式**：投资报告(50%)+期中考试(30%)+课堂表现(20%)
