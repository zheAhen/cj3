# 黄伟-运营管理

## 课程概述
本课程是工商管理专业的核心课程，系统讲解企业运营管理的理论和方法，包括流程设计、质量管理、供应链管理等内容。

## 课程特色
1. **模拟运营**：使用企业运营模拟软件
2. **工厂参观**：实地考察制造企业运营
3. **案例竞赛**：分组解决实际运营问题
4. **考核方式**：案例分析(40%)+期中考试(30%)+期末报告(30%)
