# 陈敏-广告学

## 课程概述
本课程是传播学专业的重要课程，研究广告的理论和实践，包括广告策划、创意设计、媒体投放、效果评估等内容。

## 课程特色
1. **创意设计**：完成广告创意作品
2. **案例分析**：研究经典广告案例
3. **行业实践**：参观广告公司
4. **考核方式**：广告策划(60%)+期中考试(20%)+课堂展示(20%)
