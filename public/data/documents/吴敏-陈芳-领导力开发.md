# 吴敏-陈芳-领导力开发

## 课程概述
本课程是MBA项目的重要课程，研究领导力的理论和实践，包括领导风格、团队建设、变革管理等内容。

## 课程特色
1. **360度评估**：进行领导力测评
2. **角色扮演**：模拟领导情境
3. **案例研究**：分析领导力案例
4. **考核方式**：领导力发展计划(50%)+期中考试(30%)+团队表现(20%)
