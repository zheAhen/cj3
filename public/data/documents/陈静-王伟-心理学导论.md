# 陈静-王伟-心理学导论

## 课程概述
本课程是心理学专业的入门课程，系统介绍心理学主要分支领域，包括认知心理学、发展心理学、社会心理学等内容。

## 课程特色
1. **跨学科视角**：结合神经科学、社会学等多学科知识
2. **实验演示**：课堂进行经典心理学实验演示
3. **小组项目**：设计并实施小型心理学研究
4. **考核方式**：小组项目(40%)+课堂参与(20%)+期末考试(40%)
