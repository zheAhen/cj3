# 全国高校课程锐评智能体

基于React的前端项目，使用Tailwind CSS和Ant Design实现课程评价平台。

## 功能特性
- 课程评价展示
- Markdown文档渲染
- 双链接功能
- 响应式设计

## 快速开始
1. 安装依赖
```bash
npm install
```

2. 启动开发服务器
```bash
npm start
```

3. 构建生产版本
```bash
npm run build
```

## 项目结构
- `src/pages/` - 页面组件
- `src/data/` - 模拟数据
- `public/` - 静态资源
